插入排序法
概念有點類似平常玩的撲克牌。當莊家發牌時，玩家拿到第一張牌、接著拿到第二張牌的時候，就會先把兩張作比較排序好。

接下來第三張牌接著插入這堆已經排序好的排組，接著同理。直到撲克牌發完的時候，手上的撲克牌也已經依照點數大小排序好了。
